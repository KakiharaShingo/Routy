# テストデータ生成ガイド

カテゴリー機能のテストに使用できるテストデータを簡単に生成できます。

## 方法1: アプリ内から生成（推奨）

最も簡単で確実な方法です。

### 手順

1. アプリを起動
2. 画面下部の「プロフィール」タブをタップ
3. 一番下までスクロール
4. **「デバッグ機能」** カードが表示されます

### デバッグ機能の内容

#### 直接データ追加（DB）
データベースに直接チェックポイントを追加します。

- **カテゴリーテストデータ作成** - 9件（各カテゴリー1件ずつ）
- **拡張テストデータ作成（20件）** - 20件（各カテゴリー複数件）

すぐに旅行リストに表示され、マップで確認できます。

#### 写真ライブラリに保存
位置情報付きの美しいテスト写真を生成してカメラロールに保存します。

- **テスト写真を生成（9枚）** - 9件（各カテゴリー1件ずつ）
- **拡張テスト写真を生成（20枚）** - 20件（各カテゴリー複数件）

保存後、旅行詳細画面の「写真から追加」で取り込むと、自動的にカテゴリーが判定されます。

## テストの流れ

### パターンA: 直接データベースに追加
1. 「カテゴリーテストデータ作成」をタップ
2. すぐに旅行リストに「カテゴリーテスト旅行」が追加される
3. タップして「地図で見る」を選択
4. カテゴリー別に色分けされたピンが表示される

### パターンB: 写真から取り込み
1. 「テスト写真を生成（9枚）」をタップ
2. 写真ライブラリへのアクセスを許可
3. 完了メッセージを確認
4. 旅行を作成または既存の旅行を開く
5. 「写真から追加」→「手動」を選択
6. 生成された写真を選択
7. 位置情報とカテゴリーが自動判定される

## 注意事項

- デバッグ機能は **DEBUGビルドでのみ表示** されます
- リリースビルドでは表示されません
- 写真生成には写真ライブラリへのアクセス許可が必要です
- 生成された写真は実際のカメラロールに保存されます

## トラブルシューティング

### 写真が生成されない
- 写真ライブラリへのアクセスを許可してください
- 設定 → プライバシー → 写真 → Routy で確認できます

### カテゴリーが表示されない
- マップでピンをタップすると、カテゴリー別の色とアイコンが表示されます
- タイムラインでもカテゴリーバッジが表示されます

### データが重複して追加される
- ボタンを複数回押すと、その都度新しいデータが追加されます
- 不要な旅行は旅行リストから削除できます
